## How to run

```
git clone https://github.com/fraigo/colyseus-example.git
cd colyseus-example
npm install
npm start
```

Open [http://localhost:2567](http://localhost:2567) in your browser.

## Deployment for prototyping

You can easily deploy Colyseus Examples on Heroku.
